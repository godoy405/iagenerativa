## 🧠 3. Engenharia de Prompts e "Cicatrizes"

Aqui documento o processo de iteração com a IA. O objetivo foi refinar as perguntas para extrair insights profundos das fontes, evitando respostas genéricas.

### 🧪 Tentativa 1: Prompt Genérico
*   **Prompt:** "O que é IA Generativa?"
*   **Resultado:** A IA deu uma definição de dicionário, muito básica e sem citar as fontes específicas.
*   **Cicatriz (Dificuldade):** A resposta foi rasa e não aproveitou o material carregado.

### 🧪 Tentativa 2: Prompt Contextualizado
*   **Prompt:** "Com base nos artigos fornecidos, quais são as 3 principais barreiras para a adoção de IA em empresas tradicionais?"
*   **Resultado:** A IA listou as barreiras citadas nos textos, mas misturou opiniões de autores diferentes sem distinguir.
*   **Melhoria:** Precisei pedir para ela separar por autor.

### 🏆 Prompt Final (Otimizado)
*   **Prompt:** "Atue como um especialista em transformação digital. Crie uma tabela comparativa citando os autores das fontes sobre os riscos de segurança no uso de LLMs. Cite a fonte exata para cada risco."
